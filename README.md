## About

Software engineer and founder at **DevBox**, working across the US, Togo, and the UK. I build end-to-end products: mobile apps, SaaS platforms, and the backends that power them. My focus is turning ambitious ideas into systems that hold up in production, with a particular interest in Artificial Intelligence, developer tooling, and physics.

- Currently building data and AI-driven platforms end to end, from API design to mobile delivery.
- Interested in collaborations on compelling engineering and AI projects.
- Reachable at **yaodavidlogan02@gmail.com** and at **[lesourcier.space](https://lesourcier.space)**.
